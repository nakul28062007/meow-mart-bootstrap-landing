# Meow Mart 🐱

A responsive landing page for a fictional cat adoption app, built as a 
Bootstrap practice project during Angela Yu's Web Development Bootcamp.

## What this is
A course exercise where I adapted the TinDog template project — replacing 
the dog-themed content with a cat adoption concept (Meow Mart). The layout 
structure follows the course guide; the branding, copy, and color scheme 
are my own additions.

## Built with
- HTML5
- CSS3
- Bootstrap 5

## Sections
- Hero with gradient background
- Features row
- Testimonial + press logos
- Pricing plans
- Footer
